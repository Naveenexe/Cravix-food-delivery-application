# Cravix - Food Delivery Application

Cravix is a Java-based food delivery web application built using **JSP, Servlets, Hibernate, and MySQL**.  
It allows users to register, log in, browse restaurants, view menu items, and place food orders.

## Tech Stack
- Java
- JSP & Servlets
- Hibernate ORM
- MySQL
- Maven
- HTML, CSS

## Features Implemented
- User Registration
- User Login
- Restaurant Listing Page
- Restaurant Menu Page
- Hibernate + MySQL Integration

## Project Structure
- `model` → Entity classes
- `dao` → DAO interfaces
- `daoimpl` → DAO implementation classes
- `controller` → Servlets
- `util` → Hibernate utility class
- `src/main/webapp` → JSP pages and frontend files

## Modules in Progress
- Cart Management
- Checkout / Place Order
- Order History
- Search & Filters

## Database Tables
- `users`
- `restaurants`
- `menu_items`
- `orders`
- `order_items`

## How to Run
1. Clone the repository
2. Import as **Maven Project** in Eclipse
3. Configure MySQL database
4. Update database credentials in `hibernate.cfg.xml`
5. Run on Apache Tomcat server
